Change individually.


html <script src="https://js.popout.tk"></script> <a id="link1" href="https://wikipedia.org">Open Wiki (popout)</a> <a id="link2" href="https://wikipedia.org">Open Wiki (new tab)</a> JavaScript p.set(document.getElementById("link1"), "popout"); p.set(document.getElementById("link2"), "new");
